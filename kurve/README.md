# Kurve (Hyprland + AGS)

This directory is now focused on the AGS port of Kurve for Arch Linux + Hyprland.

## AGS build

Use:

- `kurve/ags/kurve.js` - AGS Kurve widget
- `kurve/ags/README.md` - dependencies and run instructions
- `kurve/ags/preview.svg` - generated preview image

## Notes

- Legacy Waybar examples and services were removed.
- The AGS implementation keeps Kurve-style rendering modes and CAVA-driven data flow.
