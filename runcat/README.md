# RunCat (Hyprland + AGS)

This directory contains the AGS version of RunCat for Arch Linux + Hyprland.

## Use AGS version

Use the files in `runcat/ags/`:

- `runcat/ags/runcat.js` - widget implementation
- `runcat/ags/icons/` - sprite frames
- `runcat/ags/README.md` - setup and usage

## Notes

- The legacy Waybar snippets/services were removed from this repository.
- Keep upstream attribution for redistributed sprite assets.
